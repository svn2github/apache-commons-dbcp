1. INTRODUCTION

The DBCP package provides database connection pooling services. The following features are supported

  • DataSource and Driver interfaces to the pool.
  • Support for arbitrary sources of the underlying Connections.
  • Integration with arbitrary org.apache.commons.pool.ObjectPool implementations.
  • Support for Connection validation, expiration, etc.
  • Support for PreparedStatement pooling.
  • XML configuration.

2. DEPENDENCIES

The DBCP package is dependent upon the following external components for development and use:

  • Java Development Kit (Version 1.2 or later)
  • Java 2 Enterprise Edition (javax.sql packages only)
  • Jakarta-Commons Collections package
  • Jakarta-Commons Pool package
  • [currently, for JOCL configuration] Simple API for XML (Version 2.0 or later)
  • [currently, for JOCL configuration] A SAX 2.0 compliant XML parser (Xerces, XP, etc.)
  • JUnit Testing Framework (Version 3.2 or later) - for unit tests only, not required for deployment
